
            Common UI Design Patterns:
            
            1. Card Pattern:
            - Used for displaying grouped information
            - Should have consistent padding and shadows
            - Good for product listings, user profiles, articles
            
            2. Modal Pattern:
            - For secondary actions or additional information
            - Should have backdrop overlay
            - Include close button and ESC key handling
            
            3. Navigation Patterns:
            - Breadcrumb: Show hierarchy and current location
            - Tabs: For switching between related content
            - Sidebar: For persistent navigation options
